# 摄影学习



## 高级功能说明书：翻烂！

我的松下Lumix S5 原装自带的纸质说明书210页是简单版本，但其实还有606页的高级功能说明书pdf。

我把高级功能说明书606页打印，A4纸1面2页、正反面打印，打印机1200dpi，装订成5本。

---|---
VUE Vlog|长纪录片的剪辑。破解版才能解锁5分钟限制

LR|调整图像颜色
